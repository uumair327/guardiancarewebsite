# GuardianCare Website

### Empowering Children Through Education & Safety

A comprehensive, enterprise-level educational platform and landing page for **GuardianCare**. Our mission is to democratize child rights education, mental health awareness, and online safety protocols.

## ✨ Features

- **Educational Content**: Learn about POCSO, child rights, and mental health.
- **Interactive Quizzes**: Fun modules to ensure children learn efficiently.
- **Community Forums**: Safe zones for children and parents to interact.
- **Safe Transit**: Bus schedules and routes tailored for safe school travel.
- **Emergency Contacts**: Instant SOS triggers and important contact listings.
- **Enterprise Security**: Built securely for open-source scalability.

## 🚀 Quick Start

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Children-of-India-Foundation/guardiancarewebsite.git
   cd guardiancarewebsite
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Start the development server**
   ```bash
   npm start
   # or
   yarn start
   ```

## ☁️ Deployment

This project is prepared for enterprise Cloud Deployment and ships with a robust `firebase.json` for Firebase Hosting or Google Cloud integration. 

To deploy to Firebase Hosting:
```bash
npm run build
npx firebase deploy --only hosting
```
